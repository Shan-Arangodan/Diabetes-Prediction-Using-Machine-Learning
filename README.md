# 🧠 Diabetes Prediction System

![Python](https://img.shields.io/badge/Python-3.x-blue)
![ML](https://img.shields.io/badge/Machine%20Learning-Scikit--Learn-orange)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

A machine learning-based system to predict the likelihood of diabetes using patient health data. This project demonstrates the practical application of classification algorithms in healthcare analytics.

---

## 📌 Overview
This project aims to assist early detection of diabetes by analyzing key medical attributes such as glucose level, BMI, age, and other health indicators. It leverages multiple machine learning models to classify individuals as diabetic or non-diabetic.

---

## ⚙️ Tech Stack
- **Language:** Python  
- **Libraries:** pandas, NumPy, scikit-learn, matplotlib, seaborn  
- **Platform:** Google Colab  

---

## 🧠 Models Used
- K-Nearest Neighbors (KNN)  
- Decision Tree  
- Naive Bayes  

---

## 🔄 Workflow
1. Data Collection  
2. Data Preprocessing & Cleaning  
3. Feature Selection  
4. Model Training  
5. Prediction & Evaluation  

---

## 📊 Dataset
- **PIMA Indians Diabetes Dataset**  
- Source: UCI Machine Learning Repository  
- Contains structured medical data such as glucose, BMI, insulin levels, etc.

---

## 🚀 Features
- Multiple ML model comparison  
- Cleaned and normalized dataset  
- Structured pipeline from preprocessing to prediction  
- Easy to understand and reproducible
